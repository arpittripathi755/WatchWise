# 🎬 WatchWise

## 📖 Overview

WatchWise is a movie discovery web application that enables exploration of movies using real-time data from The Movie Database (TMDB) API. The application demonstrates core web development concepts such as API integration, dynamic content rendering, and the use of JavaScript array higher-order functions.

---

## ✨ Features

* **Search Movies** – Retrieve movies based on user input
* **Genre Filtering** – Filter movies by selected genres
* **Sorting Options** – Sort movies by rating, popularity, or release date
* **Dynamic UI Rendering** – Display movie data dynamically using JavaScript
* **Responsive Design** – Optimized for mobile, tablet, and desktop devices

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Language:** JavaScript (ES6)
* **API:** TMDB (The Movie Database)
* **Networking:** Fetch API

---

## 🔗 API Usage

The application integrates with the TMDB API to fetch and display:

* Movie titles
* Ratings and popularity scores
* Genre classifications
* Poster images

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/arpittripathi755/WatchWise.git
```

### 2. Navigate to the Project Directory

```bash
cd WatchWise
```

### 3. Add API Key

Open `script.js` and insert your TMDB API key:

```js
const API_KEY = "your_api_key_here";
```

### 4. Run the Application

Open `index.html` in your browser.
